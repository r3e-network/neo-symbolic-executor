# Neo Symbolic Executor v0.3.0 Release Notes

Release date: `2026-02-19`

## Summary

Version `0.3.0` focuses on release-grade validation and repeatability. The main addition is a complete DevPack corpus validation utility that extracts embedded test artifacts and executes analyzer runs in batch with structured reporting.

## What Is New

- Added `examples/validate_devpack_corpus.py`:
  - Extracts embedded manifest/NEF from DevPack `TestingArtifacts/*.cs`.
  - Runs `neo_sym.cli analyze` across the full corpus.
  - Produces structured outputs for operational review and CI tracking.
- Added release and operational documentation (`README.md`, `CHANGELOG.md`, this note).
- Centralized version handling:
  - `neo_sym.__version__` is now the source of truth for CLI version output.
